== Description ==
Appends .html extension to the Page URL when using seo permalinks and page permalink box has _html text.

== Installation ==
1. Upload pages-with-extension.php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings->Permaliks and Save for refreshing
4. Add _html on permalink box of pages you want use it
